Internal style sheet Dentro da tag head do nosso html, podemos criar uma folha de estilo interna utilizando a tag . Para isso utilizamos seletores. Seletor é o nome da tag que deseja formatar: p { color: red; } CLASSES E IDS A classe é utilizada para identificar um grupo de "indivíduos", já o ID é único e identifica um "indivíduo" específico Classe: O ponto define uma classe, o nome pode ser criado. Ex. .azul{ color: blue; } Para utilizar a classe no html, devemos adicionar class="azul" dentro da tag Identificador: O # define um identificador, o nome pode ser criado. Recomenda-se utilizar para estruturar o site Ex. #principal { background: grey; } DIV E SPAN (Containers genéricos) Div é utilizada para separar, organizar e aplicar estilos aos elementos da tela.
